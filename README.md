## 🎮 Bounce Ball Game (Tkinter)

This is a simple arcade-style Bounce Ball game developed using Python's Tkinter library.

### 🚀 Features
- Paddle controlled using keyboard
- Ball bouncing physics
- Brick breaking system
- Score tracking
- Pause functionality
- Win and Game Over screens

### 🎮 Controls
- ⬅ Left Arrow → Move paddle left  
- ➡ Right Arrow → Move paddle right  
- ⏎ Enter → Start game  
- Space → Pause / Resume  

### 🧠 How It Works
- The ball bounces around the screen
- Break all bricks to win
- If the ball hits the bottom → Game Over

### 🛠️ Technologies Used
- Python
- Tkinter
